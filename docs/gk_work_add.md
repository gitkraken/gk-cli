## gk work add

Add a git repository to the current active work item

### Usage
```
gk work add [flags]
```
### Aliases
```
add
include
```

### Examples
```
gk work add .
gk work add </home/user/code/repo>
```

### Flags

```
-h, --help   help for add
```
