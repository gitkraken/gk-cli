## gk issue

This command is integrated with your provider issue tracker and provides a convenient way to interact with issues.

Available providers: github, gitlab, jira

### Usage
```
gk issue [command]
```

### Available Commands
```
list        Show list of issues
```

### Flags

```
  -h, --help   help for issue
```
