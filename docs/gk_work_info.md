## gk work info

Displays the current info of the active work item or a specified work item. It provides an overview of the work item's' metadata, progress, and associated tasks..

### Usage
```
gk work info [flags]
```

### Aliases 
```
info 
status
```

### Examples
```
gk work info
gk work info <name>
```

### Flags

```
-h, --help   help for info
```
